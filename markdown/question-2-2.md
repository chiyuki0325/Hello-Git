# Markdown 中的数学公式

Markdown 中可以使用 LaTeX 语言书写公式。由于之前并没有学习 LaTeX，因此现场学习，并简单记录下此文档。

我使用的编辑软件是 Typora。在 Typora 的偏好设置中启用「Markdown - 内联公式」即可书写行内公式。

### 上下标

$$
a^3 + b^3 = (a+b)(a^2-ab+b^2)
$$

```latex
a^3 + b^3 = (a+b)(a^2-ab+b^2)
```
